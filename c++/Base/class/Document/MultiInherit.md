####多重继承:
    1.子类只overwrite了父类的f()函数
    2.子类还有一个g1()函数
    3.每个类都有一个自己的成员变量

####![](../image/OMemory2.jpg)
    1.每个父类都有自己的虚表
    2.子类的成员函数被放到第一个父类的表中
    3.内存布局中,其父类布局依次按声明顺序排列
    4.每个父类的虚表中的f()函数都被overwrite成了子类的f() ----解决不同的父类类开的指针指向同一个子类实例.而能够调用到实际的函数
