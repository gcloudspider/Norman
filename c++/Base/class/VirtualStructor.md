####虚构造函数
    没有虚构造函数原因:
        1.虚拟调用是一种能够在给定信息不完全情况下工作的机制。
        2.特别地虚拟允许我们调用某个函数,对于这个函数,仅仅知道它的接口,而不知道具体对象类型.
        3.但要建立一个对象,必须拥有完全信息。需要知道要建立对象的具体类型.
        所以对构造函数调用不可能是虚拟的
