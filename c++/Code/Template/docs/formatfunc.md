####函数模板格式:
    template <class 形参名,class 形参名, ....> 返回类型 函数名(参数列表)
    {
        函数体
    }

    模板形参不能为空,一旦声明了模板函数就可以用模板函数的形参名声明类中的成员变量和成员函数
    即可以在该函数中使用内置类型的地址都可以使用模板形参名

    不能在函数调用的参数中指定模板形参的类型,对函数模板的调用应使用实参推演来进行
