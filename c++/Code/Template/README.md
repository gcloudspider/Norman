####C++模板:
    模板是C++支持参数化多态的工具,使用模板可以使用户为类或者函数声明一种一般模式,使得类中的某些数据成员或成员函数的参数,返回值取得任意类型
    模板是一种对类型进行参数化的工具;

####形式:
    1.函数模板:针对仅参数类型不同的函数
    2.类模板:针对仅数据成员和成员函数类型不同的类

####注意:
    模板声明或定义只能在全局,命名空间或类范围内进行
    不能在局部范围,在函数内进行(如不能在main函数中声明或定义一个模板)
