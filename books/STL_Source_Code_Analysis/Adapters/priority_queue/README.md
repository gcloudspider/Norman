####优先级队列priority_queue:
    1.priority_queue拥有优先级的queue 
    2.容器内元素并不是根据加入顺序排列。而是根据用户自定义优先级进行排序
    3.priority_queue只能在队列尾部加入元素,在头部取出元素
    4.不能遍历容器,不需要设置迭代器
    5.默认基于vector容器作为底部结构(也可自定义指定容器) <stl_queue.h>
