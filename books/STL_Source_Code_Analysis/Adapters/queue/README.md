####容器配接器queue:
    1.queue是一种先进先出的数据结构
    2.可以对两端进行操作,但是只能在队列头进行移除元素,在队列尾部新增元素
    3.可以访问队列尾部和头部元素，不能遍历容器
    4.queue不需要自己的容器.
    5.queue默认基于deque容器作为底部结构<stl_queue.h>
    6.用户也可以自定义容器类型作为底部结构

####queue容器配接器:
    基于deque底层容器
    对外接口函数调用deque相对应方法
