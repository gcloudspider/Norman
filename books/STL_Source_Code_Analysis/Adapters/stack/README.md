####stack栈:
    1.stack是一种"先进后出"的数据结构
    2.只能在栈顶对数据进行操作,即只能在栈顶进行新增元素 移除元素,取得最顶端元素.
    3.不能进行遍历行为,所以不需要迭代器
    4.SGI STL基于deque容器作为底部结构(也可以自定义) <stl_stack.h>

####stack容器配接器:
    基于其他容器进行操作的。
    对外的接口默认调用deque接口
