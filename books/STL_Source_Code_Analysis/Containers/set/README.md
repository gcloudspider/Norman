####Set容器:
    1.set基本上是在RB-Tree基础上
    2.RB-Tree作为底层实现机制.
    3.set的操作RB-Tree都能实现
    4.set容器键值key和实值value是相同的。且在容器里面元素是根据元素的键值自动排序的
    5.不能修改set容器元素值
    6.set迭代器采用RB-Tree的const_iterator 不允许用户对其进行修改操作
    7.SGI STL 源码<stl_set.h>文件
