####模板方法:
    模板方法定义一个操作中的算法骨架,而将一些步骤延迟到子类中。

    模板方法使得子类可以不改变一个算法的结构即可重定义该算法的某些特定步骤。

####场景1:
    简历:在父类中定义操作算法骨架,而具体实现由子类完成

####场景2:
    准备饮料,泡茶和泡咖啡
