####代理模式:
    为其他对象提供一种代理以控制对这个对象的访问.

    四种情况:
        1.远程代理
        2.虚代理
        3.保护代理
        4.智能引用


####场景1:(Image 属于虚代理)
    文档中嵌入图形对象(创建图形开销大,打开文档必须迅速)
    使用代理模式,打开文档时不打开图形,而是打开图形对象的代理以替代真实图形。待到真正需要打开图形时,仍由代理负责打开
