####抽象工厂模式:
    工厂方法模式缺点:
        每增加一种产品,就需要增加一个对象的工厂,如果产品很多就要开设相应的新工厂.
        C++实现中,就是要定义一个个的工厂类,相比简单工厂模式,工厂方法需要更多类定义
