####脉冲宽度调制:
    整流 逆流 高电平修改成低电平(改变频宽)
    修改脉冲宽度 TCON寄存器改变频宽

####死区:
    高电平切换低电平时延迟一段时间。以避免功率元件烧毁。这个延迟时间就是死区

####DSP里PWM死区:
    整流逆变过程中,高低电平不能同时导通.否则电源会段路。理论上DSP产生PWM 不会同时通。但器件原因PWM不可能是瞬时电平跳变，以梯形下降。

####使能:
    负责控制信号的输入和输出叫使能-----"允许信号"

b.设定TCNTB0,TCMPB0，;

d.启动定时器并选择是否自动加载和初始化电平;
